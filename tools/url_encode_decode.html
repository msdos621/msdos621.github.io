Input any string of text and press encode or decode in order to url encode / decode the text.

Encode
Decode

Here is the code used to do this

{% highlight javascript %} {% raw %} $(document).ready(function () { $('#encode').click(function() { $('#decode_box').val(encodeURIComponent($('#decode_box').val())); }); $('#decode').click(function() { $('#decode_box').val(decodeURIComponent($('#decode_box').val())); }); }); {% endraw %} {% endhighlight %}
